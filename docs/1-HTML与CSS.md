# 前端 第一轮考核

## 目的

HTML与CSS的基本语法

- HTML标签,标签的语义化（HTML5新特性）
- CSS 盒模型，利用 CSS 完成简单的布局 （水平/垂直居中的实现）
- CSS 定位，实现部分元素的绝对定位 
- 熟悉浏览器 F12 开发者工具的使用，利用它检查元素 
- 掌握简单的设计稿还原

## 任务

将设计图.png 文件还原成 HTML/CSS 文件（包括导航条、文字）

1. 通过自己的理解，将设计稿分解成合理的部件，网页布局合理
2. 使用**外联css**
3. 在人物介绍板块实现鼠标移入、移出效果
4. 导航栏应能点击跳转至相应位置
5. HTML/CSS 书写规范，命名规范
6. 不允许使用图形化工具制作界面
